# Video AI Analysis - 前端界面

🎬 **Apple风格 + Premiere Pro时间轴** 的视频分析界面

## ✨ 界面特性

### 🍎 Apple UI 设计
- **极简黑色主题**：灵感来自Final Cut Pro X
- **毛玻璃效果**：半透明导航栏
- **流畅动画**：所有交互都有过渡效果
- **SF字体系统**：使用Apple的San Francisco字体

### 🎞️ Premiere Pro 时间轴
- **多轨道显示**：视频、运镜、光线、调色分别为独立轨道
- **直观片段**：每个特征显示为彩色片段
- **置信度显示**：通过数字显示分析置信度
- **时间标尺**：精确到毫秒的时间显示

## 📖 使用指南

### Learn 模式（单视频分析）

1. **选择Learn模式**
   - 点击顶部导航栏的"Learn"标签

2. **上传视频**
   - 点击"目标视频"卡片
   - 选择本地MP4文件
   - 或拖拽文件到上传区域

3. **调整参数**
   - 抽帧率：控制分析精度（建议0.5-2 fps）
   - 最大帧数：控制成本（建议10-30帧）

4. **开始分析**
   - 点击"✨ 开始分析"按钮
   - 等待处理（通常10-30秒）
   - 查看时间轴结果

### Compare 模式（双视频对比）

1. **选择Compare模式**
   - 点击顶部导航栏的"Compare"标签

2. **上传两个视频**
   - 上传目标视频（参考）
   - 上传用户视频（待改进）

3. **查看对比结果**
   - 查看改进建议
   - 对比两个视频的特征差异

## 🎨 时间轴说明

### 轨道类型

| 轨道 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 视频片段 | 🎬 | 紫色 | 镜头切分结果 |
| 运镜 | 📹 | 绿色 | 推拉摇移等运镜方式 |
| 光线 | 💡 | 橙色 | 布光方式和光线特点 |
| 调色 | 🎨 | 粉色 | 色彩风格和调色方案 |

### 交互操作

- **点击片段**：查看详细信息
- **+ / −按钮**：缩放时间轴
- **导出结果**：下载JSON格式的分析结果

const API_BASE_URL = 'http://localhost:8000';

## 🐛 故障排除

### 问题：无法上传文件

**原因**：文件路径配置
**解决**：修改 `js/app.js` 中的路径逻辑

### 问题：时间轴不显示

**原因**：API返回数据格式
**解决**：检查浏览器控制台的错误信息

### 问题：样式显示异常

**原因**：浏览器兼容性
**解决**：使用Chrome、Safari或Edge浏览器
